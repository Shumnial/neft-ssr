- [Vue](https://vuejs.org/v2/api/)
- [NuxtJS](https://nuxtjs.org) - universal mode

### Кладбище ссылок

## Build/Dev

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
